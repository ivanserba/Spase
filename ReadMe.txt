   Описание программы:
Это инженерный калькулятор, созданный для вычисления сложных математических выражений.
Он поддерживает все известные алгебраические функции (их полный список - в файле "functions.txt")
Также, возможен ввод чисел в 4 системах исчисления (например, 0b11101 - двоичная, 0o3456 - восьмеричная,
96484 - десятичная, 0хААВ4 - шестнадцатиричная). Если при вводе выражения будет ошибка, то предупреждение 
о ней будет выведено в выходной файл.
   Правило пользования:
1. Поместить в фолдер проекта все файлы (код, "functions.txt", "input.txt").
2. Открыть файл "input.txt".
3. Ввести свои выражения для вычислений (каждое выражение заканчивается знаком '='), количество выражений неограничено.
4. Скомпилировать и запустить код.
5. Проверить в фолдере проекта появившийся файл "outputs.txt".
6. Посмотреть его содержимое.
